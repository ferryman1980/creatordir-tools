# CreatorAI Tools 🚀

> **The Ultimate AI Tools Directory for Content Creators**
> [creatordir-tools.vercel.app](https://creatordir-tools.vercel.app)

## 📋 Overview

CreatorAI Tools is a curated directory of **378+ AI tools** for content creators. We review, compare, and rank AI tools across writing, design, video, audio, and productivity categories.

## ✨ Features

- 🎯 **217+ AI tool detail pages** with honest reviews
- 📝 **99+ English articles** covering comparisons, guides, and tutorials
- 🔍 **Built-in search** across all tools and articles
- 📊 **Honest pricing** - free tiers, freemium, and paid options
- 🔗 **Affiliate links** that help support the project
- 📱 **Responsive design** - works on all devices
- 🚀 **Vercel-hosted** for fast global delivery

## 🚀 Quick Start

This is a static HTML site. To run locally:

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/creatordir-tools.git

# Open in browser
open index.html
# Or serve locally
npx serve .
```

## 🌐 Deployment

Deployed on [Vercel](https://vercel.com) for free.

```bash
vercel --prod
```
